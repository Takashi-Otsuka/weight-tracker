# 体重管理アプリ 仕様書（MVP）

## 1. 概要

### 目的

日々の体重記録を簡単に行い、体重推移の可視化と目標管理を支援する。

ユーザーが以下を簡単に把握できることを目的とする。

* 現在の体重推移
* 目標体重との差
* 目標達成に必要な減量ペース
* 現在のペースでの達成予測

## 10. シミュレーション機能

### パターンA

入力

* 現在体重
* 目標体重
* 目標日

計算

```text
必要減量量 = 現在体重 - 目標体重

必要日数 = 目標日 - 今日

日あたり減量量
週あたり減量量
月あたり減量量
```

### パターンB

入力

* 現在体重
* 目標体重
* 日あたり減量量

計算

```text
必要日数 = 必要減量量 ÷ 日あたり減量量

達成予定日 = 今日 + 必要日数
```

## 11. 健康ペース判定

週あたり減量率を利用する。

### 判定

* 0.5%未満 → ゆるやか
* 0.5〜1.0% → 推奨
* 1.0〜1.5% → やや速い
* 1.5%以上 → 見直し推奨
